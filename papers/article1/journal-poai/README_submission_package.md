# POAI Submission Package (Draft)

Prepared for journal: Problems of Artificial Intelligence (PAI)
Instructions page: http://paijournal.guiaidn.ru/en/instructions.html

## Included files

- `templates/paper_template.doc` - official paper template from journal site
- `templates/covering_letter_template.doc` - official covering letter template
- `templates/personal_info_form_template.doc` - official personal information form
- `templates/resume_template.doc` - official resume template
- `manuscript_poai_clean.txt` - cleaned, template-ready manuscript text with normalized POAI section structure
- `manuscript_poai_clean.docx` - editable DOCX version of the cleaned manuscript
- `manuscript_poai_draft.txt` - journal-adapted manuscript draft content (ready to paste into Word template)
- `cover_letter_draft_ru.txt` - draft covering letter text (RU)
- `personal_information_form_draft_ru.txt` - draft personal information form content (RU)
- `external_review_draft_ru.txt` - draft of the required external review document
- `author_end_information_block_ru_en.txt` - author-information block for the end of the article
- `reference_policy_audit_poai.md` - bibliography compliance audit against journal policy
- `pai_internal_citation_candidates.md` - candidate in-journal references to review
- `license_and_copyright_notes_poai.md` - notes on license agreement and copyright paperwork
- `submission_checklist_poai.md` - compliance checklist and remaining actions

## Important notes

1. Journal requires final manuscript in Word format according to template (`.doc` template downloaded).
2. Journal requires multilingual blocks (RU, EN, UKR) for title/author/affiliation/annotation.
3. Journal requires detailed resume in EN and RU (0.5-1 page each).
4. Journal requires certified external review from a detached organization.
5. Journal requires printed signed copy and identical electronic version.
6. Journal provides a separate license agreement form that should be kept ready.

## Fast finalization workflow

1. Open `templates/paper_template.doc` in LibreOffice/Word.
2. Use `manuscript_poai_clean.docx` or `manuscript_poai_clean.txt` as the primary source for template insertion.
3. Open `templates/covering_letter_template.doc` and paste/finalize text from `cover_letter_draft_ru.txt` on organization letterhead.
4. Open `templates/personal_info_form_template.doc` and fill with `personal_information_form_draft_ru.txt`.
5. Finalize `external_review_draft_ru.txt` with reviewer data, signature, and seal.
6. Prepare the license agreement and end-of-article author information block.
